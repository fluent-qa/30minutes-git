## Git 基本命令

* 1. 创建GIT仓库:
  - git init
  - git clone 
  - git remote add origin <remote-url>

* 2. 修改内容
  - git add 
  - git commit
  - git pull
  - git push
  - git merge
  - git checkout
  - git reset

![](git-modify.png)


## Git Branch 分支管理

- git branch 
- git checkout
- git checkout -b feat/remote-branch
- git pull origin feat/remote-branch
